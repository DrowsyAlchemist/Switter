# AuthService

Микросервис аутентификации и авторизации для платформы Switter. Обеспечивает регистрацию пользователей, вход в систему, управление JWT токенами и интеграцию с другими сервисами через Kafka.

##  Функциональность

### Основные возможности
- **Регистрация пользователей** - создание новых учетных записей
- **Аутентификация** - вход в систему по email/username и паролю
- **JWT токены** - генерация и валидация access/refresh токенов
- **Обновление токенов** - получение новой пары токенов по refresh token
- **Отзыв токенов** - принудительное завершение сессий
- **Валидация паролей** - безопасное хеширование с BCrypt
- **Health checks** - мониторинг состояния сервиса и зависимостей

### Технические особенности
- Entity Framework Core с миграциями
- Redis для хранения refresh токенов
- Kafka для интеграционных событий
- JWT с настраиваемыми параметрами
- Structured logging
- Docker контейнеризация
